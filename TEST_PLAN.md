# Программа тестирования

## Функциональные тесты

### 1. Загрузка страницы
**Ожидаемый результат:** Отображается экран приветствия с категориями и полем ввода имени
- [ ] Все 10 категорий отображаются с иконками
- [ ] Поле ввода имени работает
- [ ] Кнопка "Начать квиз" неактивна без имени

### 2. Начало игры
**Ожидаемый результат:** После ввода имени отображается игровое поле
- [ ] Игровое поле показывает все категории и ячейки с очками
- [ ] Счёт игрока отображается в хедере
- [ ] Отображается прогресс (0/15)

### 3. Выбор вопроса
**Ожидаемый результат:** При клике на ячейку открывается карточка вопроса
- [ ] Отображается категория и стоимость вопроса
- [ ] Для категории "У нас проблемы с текстом" показываются неправильные слова
- [ ] Для остальных категорий — подсказка
- [ ] Кнопка "Показать ответ" работает

### 4. Ответ на вопрос
**Ожидаемый результат:** После нажатия "Показать ответ" отображается правильный ответ
- [ ] Название песни и исполнитель отображаются корректно
- [ ] Кнопки "Угадал(а)" и "Не угадал(а)" работают
- [ ] Очки начисляются при правильном ответе

### 5. Переход к следующему вопросу
**Ожидаемый результат:** После ответа отображается результат и кнопка "Следующий вопрос"
- [ ] При верном ответе — зелёный фон и +очки
- [ ] При неверном — красный фон
- [ ] Кнопка возвращает к игровому полю
- [ ] Отвеченный вопрос отмечен на поле

### 6. Окончание игры
**Ожидаемый результат:** После 15 вопросов отображается экран результатов
- [ ] Итоговый счёт отображается
- [ ] Статистика (верно/неверно/всего)
- [ ] Прогресс-бар с процентом
- [ ] Кнопка "Играть ещё раз"

## Пользовательские тесты

### Адаптивность
- [ ] На мобильных устройствах (320px) интерфейс читаем
- [ ] На планшетах (768px) все элементы доступны
- [ ] На десктопах (1920px) используется максимальная ширина

### Производительность
- [ ] Страница загружается менее чем за 3 секунды
- [ ] Анимации плавные (60fps)
- [ ] Нет задержек при кликах на кнопки

### Доступность
- [ ] Все кнопки имеют понятные подписи
- [ ] Цветовые индикаторы дублируются текстом
- [ ] Шрифты достаточно крупные для чтения

## Тестирование данных

- [ ] Все 75 вопросов загружаются из Supabase
- [ ] Категории отсортированы по имени
- [ ] Вопросы сгруппированы по категориям
- [ ] Нет дублирования вопросов в одной категории с одинаковыми очками
