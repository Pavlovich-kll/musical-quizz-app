# Музыкальный квиз

Веб-приложение музыкальной викторины на основе презентации "Музыкальный квиз.pptx".

## Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/Pavlovich-kll/musical-quizz-app.git
cd musical-quizz-app

# Установить зависимости
npm install

# Настроить переменные окружения
cp .env.example .env.local
# Заполнить NEXT_PUBLIC_SUPABASE_URL и NEXT_PUBLIC_SUPABASE_ANON_KEY

# Запустить в режиме разработки
npm run dev
```

## Настройка Supabase

1. Создать проект в [Supabase](https://supabase.com)
2. Получить URL проекта и anon key
3. Заполнить `.env.local`:
   ```
   NEXT_PUBLIC_SUPABASE_URL=https://ваш-проект.supabase.co
   NEXT_PUBLIC_SUPABASE_ANON_KEY=ваш-anon-key
   ```
4. Выполнить миграции из `supabase/migrations/`

## Деплой на Vercel

1. Подключить GitHub репозиторий к Vercel
2. Добавить переменные окружения:
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
3. Деплой происходит автоматически при пуше в main

## Категории квиза

| Категория | Описание |
|-----------|----------|
| Ё май онли ван | Угадай песню по забавному переводу |
| Потеряли слова | Вспомни пропущенные слова |
| Я сижу на камушке ла-ла-ла | Песни с "ла-ла-ла" |
| У нас проблемы с текстом | Неправильные слова песни |
| Янсеп | Угадай песню по фрагменту |
| Сей на-на-на | Песни с "на-на-на" |
| ООО «Мы не придумали» | Оригинал или кавер |
| Еврибади лавс самбади | Песни для вечеринок |
| Соединились наши орбиты/планеты | Миксы из двух песен |
| Чай не утопнем | Необычные песни |

## Правила игры

- Игрок выбирает категорию и стоимость вопроса (10-50 очков)
- Открывается вопрос: показывается подсказка или неправильные слова
- Игрок нажимает "Показать ответ" и честно оценивает: угадал или нет
- За правильный ответ начисляются очки
- Всего 15 вопросов в раунде
